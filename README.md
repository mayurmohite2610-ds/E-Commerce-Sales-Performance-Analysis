## Project Description

This project analyzes e-commerce sales data using Data Analytics techniques. The analysis helps identify sales trends, top-performing products, customer purchasing behavior, and regional sales performance.

---

## Objectives

- Analyze monthly sales trends.
- Identify top customers.
- Analyze region-wise sales performance.
- Identify best-selling products.
- Create interactive dashboards for data visualization.

---
